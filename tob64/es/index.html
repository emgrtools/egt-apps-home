ToB64
ToB64 es bastante simple y fácil de usar. ToB64 tiene dos herramientas muy útiles: Codificador y Decodificador.

La herramienta Codificar te permite convertir imágenes comunes a código base64. Sólo tienes que elegir una imagen de tu dispositivo. Una vez convertida se mostrará en 3 formatos diferentes: código base64 simple, código HTML y código CSS. Por último, puedes copiar los resultados pulsando el botón Copiar.

Formatos soportados: .png, .jpg y .jpeg.

La herramienta Decodificar te permite convertir códigos base64 en imagen. Simplemente pega un código base64 completo y válido, y pulsa Decodificar. También puede guardar la imagen generada en tu dispositivo.
Disponible para Windows 10
